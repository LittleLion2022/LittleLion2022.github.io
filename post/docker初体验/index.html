安装Docker(Ubuntu 2021 64位)

执行命令：

1

sudo apt-get install -y docker.io

启动Docker

1

sudo systemctl start docker

输入docker version查看Docker的版本：

注意到：这里只显示了Client的信息，下面有一个报错: persission denied…，这个是因为我们安装的时候是用的sudo安装，在这里是没有权限连接docker的服务端，解决办法是把当前用户加入到docker组里面去。

先尝试增加一个docker组：

1

sudo groupadd docker

把当前用户添加到docker组：

1

sudo gpasswd -a ${USER} docker

重新启动Docker:

1

sudo service docker restart

（第一次实际操作时我不小心敲成restart docker了）

切换当前会话到docker组：

1

newgrp - docker

再次输入docker version，正常显示完整信息：
